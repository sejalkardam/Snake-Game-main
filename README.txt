To set up the game please ensure the following points:-

-Make sure you have installed pygame library on your device. (If not, Go to Command Prompt/Powershell/Terminal and type "pip install pygame"

- Now run this "Snake_game_main.py" file in your python code editor

- Make sure that you do not delete ANY of the files or else the game may crash.

-Also, pls ensure that the file structure is not disturbed (the files shall remain in the same folders as they are in this repository.)

- The instructions on how to play the game can be found in the "How to Play" section of the game itself.
